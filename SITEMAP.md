User Journey Examples:

1. LEARN ABOUT COMPANY:
   Home → About → Services → Contact

2. SCHEDULE VISIT:
   Home → Book a Visit
   Services → Request Quote → Book a Visit
   Events → Register → Book a Visit (pre-filled)

3. READ CONTENT:
   Home → News → Article
   Home → Blog → Post

4. EXPLORE CAPABILITIES:
   Home → Services → Individual Service → Request Quote

5. CONTACT:
   Any Page → Header/Footer → Contact
   Any Page → "Book a Visit" CTA → Booking

• Reduced spacing

MOBILE (< 768px)
• Hamburger menu
• Single column layouts
• Stacked elements
• Touch-optimized buttons
