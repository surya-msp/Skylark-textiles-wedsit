📞 GETTING STARTED
═══════════════════════════════════════════════════════════════

STEP 1: Install Dependencies
$ npm install

STEP 2: Start Development Server
$ npm run dev
→ Opens at http://localhost:5173

STEP 3: Review Documentation
• Read QUICKSTART.md first
• Then CONTENT_GUIDE.md for PDF integration

STEP 4: Add Content
• Search for "TODO:" in codebase
• Update JSON files in src/data/
• Replace placeholder text in components

STEP 5: Add Images
• Place images in public/assets/
• Follow naming convention in CONTENT_GUIDE.md

STEP 6: Test
• Test all pages
• Test all forms
• Test mobile responsiveness

STEP 7: Build & Deploy
$ npm run build
• Review DEPLOYMENT_CHECKLIST.md
• Deploy dist/ folder
